Background Of Phishing

Phishing has become one of the most common cyberattacks today. Attackers launch phishing attacks now in a variety of ways, like spoof calls, messages, social networking, and emails, with the intention of obtaining sensitive information from victims. The most common way victims fall for phishing is by clicking on a fraudulent link in an email that was sent by an attacker. This is called email phishing. This link will take victims to websites that appear legitimate but are run by the attackers. Attackers construct illegal email accounts using real company details and send the email to victims, impersonating a real person, while making the victim click on the link to the fraud website using social engineering. If a person clicks on the link and visits the site and tries to login or perform any action, then the attackers can get passwords, login credentials, credit card information, and other sensitive information.

Research Objectives

  • To develop a phishing website detection model using Graph Neural Networks (GNN)-based approach using visual similarity features to identify between legit and phishing websites.
  • To develop a phishing website detection model using approach using website features to identify between legit and phishing websites.
  • To develop a phishing email detection model using naïve byes with sentiment analysis.
  • To Develop an adaptive machine learning model for URL-based phishing website detection within a browser extension. Focus on feature extraction and usability.

Research Main objective

To develop a mobile application and a web browser extension that is capable of classifying phishing websites from legitimate websites powered by machine learning and deep learning models to provide the user with warnings regarding any potential risk of being navigated into a phishing website while preserving the privacy of the user, and if the user gets a phishing email, warn the user before the user clicks on any links or is misled by false information.

Research Problem

  • How to use visual similarity to detect phishing websites?
  • How to use the website features to detect phishing websites?
  • How to identify phishing email by the header and the text in the mail?
  • How to detect phishing websites focusing on the URL of the suspicious website?
  • How to inform a user about a phishing email or a phishing website through a browser extension and a mobile app?

Research Abstract

The integrated system consists of an extension and an Android mobile application. The mobile application allows users to input images and check if they are phishing sites or emails. It uses a graph neural network model for image input and a naïve bayes model for email checker. The browser extension complements the mobile application, deploying machine learning models to detect phishing websites based on various features. Real-time analysis provides users with immediate feedback, enabling informed decisions about the sites they interact with. This comprehensive solution uses machine learning methodologies and a Random Forest Classifier to proactively identify and safeguard against phishing threats across multiple platforms.
